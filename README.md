# README - Event Booking & Management System (Laravel + WordPress API Integration)

**Description:**
This is a **starter template** for an Event Booking & Management System. It demonstrates event creation, booking functionality, and API integration with WordPress for event listing.

**Important:**
This project is a **template** and **requires customization** based on specific client requirements. Features, styling, and integrations should be adapted as needed.

---

## Key Features

* Event CRUD and booking functionality
* Booking management with dummy data
* Blade templates for event listing and bookings
* API routes for WordPress integration
* Seeder with sample event for demo purposes

---

## Setup Instructions

1. Clone the repository.
2. Run `composer install`.
3. Run `npm install && npm run dev`.
4. Set up `.env` with database credentials.
5. Run migrations and seeders: `php artisan migrate --seed`
6. Start development server: `php artisan serve`

---

## Notes

* This project is a **demo template** and not fully production-ready.
* Payment gateways, booking validation, and styling require customization based on client needs.
* Sample events can be replaced with client-specific events and data.

---

## Technologies Used

Laravel 10, MySQL/PostgreSQL, Blade Templates, PHP, HTML/CSS, API Integration
